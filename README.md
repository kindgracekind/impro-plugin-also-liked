# impro-plugin-also-liked

Have you ever wondered "what else is like this?" Wonder no more!

This plugin shows posts that were liked by the same people in a section underneath each post, powered by the For You [Also Liked API](https://foryou.club/also-liked).

This plugin was heavily inspired (i.e. copied) from the same feature in [Witchsky](https://tangled.org/jollywhoppers.com/witchsky.app)- leave them a star on Tangled!
